# Governance Alignment Simulation

This repository contains the simulation code and experimental results for a study examining how governance architectures influence alignment and coordination outcomes in multi-agent AI systems.

The project implements an agent-based simulation in which learning agents interact under different governance regimes. These regimes vary institutional features such as incentives, communication, memory, and enforcement mechanisms. The simulation evaluates how these governance structures affect coordination, reward outcomes, and system stability.
